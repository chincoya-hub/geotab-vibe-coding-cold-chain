# ❄️ Geotab Vibe: Smart Cold Chain Visualizer

This repository showcases a specialized **Geotab Add-in** for Cold Chain management, developed entirely through **Vibe Coding** with **Gemini** for the **Geotab Vibe Coding Competition 2026**.

## 📌 Project Overview
The **Smart Cold Chain Visualizer** is a high-impact dashboard designed for **Pharmaceutical and Perishable Food Fleet Managers**. 

The core problem it solves is the complexity of monitoring temperature integrity during transit. Instead of raw data logs, this Add-in provides a highly intuitive interface that allows managers to see exactly where and when temperature fluctuations occurred along a vehicle's route.

## 🤖 The Vibe Coding Journey
This project was born from a collaborative dialogue between human intent and AI execution. 
* **The Process:** I shared the vision of a "color-coded temperature map" with **Gemini**. 
* **The Iteration:** Through several versions, the AI translated my conceptual requirements into valid Geotab JSON structures. 
* **The Goal:** Achieving a balance between technical Geotab SDK constraints and a user-friendly "vibe" that satisfies the modern fleet manager's needs.

## 🌟 Key Features
* **Visual Temperature Dashboard:** A clean, easy-to-understand interface that replaces complex spreadsheets.
* **Color-Coded Route Analysis:** Real-time visual cues (colors) that change based on temperature thresholds, making it easy to spot risks at a glance.
* **Daily Fleet Insights:** Comprehensive data summary of the day’s journey to ensure compliance with cold chain protocols.
* **Client-Centric Design:** An attractive and professional solution that adds immediate value to the end-customer.

## 🚀 How to Install in MyGeotab
1. Log in to your **MyGeotab** database.
2. Navigate to **Administration** > **System** > **Add-ins**.
3. Click **New Add-in**.
4. Copy the content from `final_addin_config.json` and paste it into the editor.
5. Click **Save** and enjoy the new "vibe" in your fleet management.
